# CallMask — Política de Privacidade

Este repositório existe apenas para **hospedar a política de privacidade** do aplicativo
Android **CallMask** (`com.juzoluis.callmask`), exigida pela Google Play por causa da
permissão `READ_CALL_LOG`. O código-fonte do aplicativo não está aqui.

- Português: <https://juzoluis.github.io/callmask-privacy/>
- English: <https://juzoluis.github.io/callmask-privacy/privacy-en.html>

As páginas são servidas por GitHub Pages a partir da raiz do branch `main`.
A versão de trabalho vive no repositório do app, em `docs/`, e é copiada para cá.
